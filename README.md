# Mandolin

### Dependencies

 * [Haxe](https://haxe.org/)
 * [hxjava](https://lib.haxe.org/p/hxjava)
 * [hxcpp](https://lib.haxe.org/p/hxcpp)

Run `haxelib install all` to install the dependencies.

### Compile java

```
haxe build-java.hxml
java -jar bin/Mandolin.jar
```

### Compile cpp

```
haxe build-cpp.hxml
```
